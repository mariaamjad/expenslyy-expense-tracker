***Expensly – Expense Tracker Web App***

A clean and simple expense tracking web app built using vanilla JavaScript, designed to help users manage daily spending with ease.

🔗 Live Demo: https://expenslyy-expense-tracker.netlify.app/ 

**Core Functionality**
- Add expenses with amount, description, and category
- View all expenses in a structured table
- Real-time total calculation
- Delete individual expenses
- Data persistence using localStorage

**Enhanced Features**
- Filter expenses by category
- Sort expenses (e.g., Latest First)
- Clean and responsive UI
- Light mode toggle 
- Basic form validation

**🛠️ Tech Stack**
- HTML5
- CSS3
- Vanilla JavaScript (ES6)
- localStorage API
